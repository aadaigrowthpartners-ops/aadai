# Aadai Growth Partners

## Prerequisites

* Node.js 20+
* npm

## Installation

npm install

## Environment Variables

Create a `.env` file based on `.env.example`.

## Run Locally

npm run dev

## Production Build

npm run build

## Preview Build

npm run preview

## Deployment

This project is configured for Netlify deployment.

Environment variables must be configured inside Netlify Site Settings → Environment Variables.
